# Uber Data Analysis Dashboard (Power BI, DAX)

Interactive Power BI dashboard analyzing 150,000+ Uber ride bookings across 4 vehicle types with ₹31.6M total booking value

## Overview
Analyzed 150,000+ Uber ride booking records across 4 vehicle types 
(Auto, Bike, Sedan, Uber XL) with total booking value exceeding ₹31.6M.

## Business Problems Solved
- Driver & customer retention through rating-based segmentation
- Payment method strategy — UPI dominates at ~44% share (₹14.84M)
- Seasonal demand forecasting through monthly booking trends

## Key Insights
- Auto highest booking value at ₹12.88M
- Uber XL lowest at ₹1.53M
- UPI is cheapest transaction method — recommended for cashback promotion
- Peak booking month: March (₹1,157K)

## Technical Implementation
- Built 5-page drill-through dashboard with per-vehicle segmentation
- Designed DAX measures for KPI tracking across payment methods
- Implemented dynamic filters and slicers for multi-dimensional exploration
- Applied treemap and conditional formatting to surface business insights
- Used custom button navigation for seamless UX across vehicle pages

## Tools Used
Power BI, DAX, Power Query
